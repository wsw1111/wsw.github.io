解决pip下载python包很慢的问题

前言

pip是python的包管理工具，它可以让我们安装python的包，并且可以自动更新，但是pip是从pypi中下载文件的，pypi是python官方的第三方库的仓库， 用的是国外的服务器，下载速度自然很慢。

解决方案

我们只需要在pip下载时将下载源切换为国内源即可（module替换为你要下载的包名）
